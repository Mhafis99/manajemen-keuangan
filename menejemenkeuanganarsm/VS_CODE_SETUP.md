# VS Code Extensions yang Dibutuhkan untuk GitHub Development

## Install dari VS Code Extensions Marketplace

### 1. Git & GitHub
- **GitHub Pull Requests** (by GitHub)
  - Review PR langsung dari VS Code
  - Merge, approve, request changes

- **GitLens** (by GitKraken)
  - Melihat blame (siapa yang mengubah baris kode)
  - History file yang detail
  - Compare branches

- **GitHub Copilot** (opsional)
  - AI assistant untuk coding

### 2. Next.js / TypeScript
- **TypeScript Vue Plugin (Volar)** sudah ada biasanya
- **ESLint** sudah ada

### 3. Productivity
- **Prettier** - Code formatter
- **Auto Rename Tag** - Rename tag otomatis
- **Bracket Pair Colorizer** - Warna bracket berbeda
